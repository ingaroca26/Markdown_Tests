# Python Notes

## `None`

### Default Value

~~~python
data: None = None
print(data)

~~~

~~~text
None
~~~

### Constructor

~~~python
information: None = type(None)()
print(information)

~~~

~~~text
None
~~~

### `type()`

~~~python
details: None = None
details_type = type(details)
print(details_type)

~~~

~~~text
<class 'NoneType'>
~~~

## `bool` Default Value

~~~python
available: bool = False
print(available)

~~~

~~~text
False
~~~

## `bool` Constructor

~~~python
accessible: bool = bool()
print(accessible)

~~~

~~~text
False
~~~
