# Python Notes

## `None` Default Value

~~~python
data: None = None
print(data)

~~~

~~~text
None
~~~

## `None` Constructor

~~~python
information: None = type(None)()
print(information)

~~~

~~~text
None
~~~

## `None` `type()`

~~~python
details: None = None
details_type = type(details)
print(details_type)

~~~

~~~text
<class 'NoneType'>
~~~
